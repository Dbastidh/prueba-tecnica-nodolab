# Prueba Técnica Maquetación - NodoLab

Este repositorio contiene la solución a la prueba técnica de maquetación HTML y SCSS, diseñada bajo las especificaciones requeridas para entornos virtuales (Moodle).

## 🌐 Vista previa en vivo
Puedes ver el resultado visual desplegado en vivo directamente en internet a través de GitHub Pages aquí:
https://dbastidh.github.io/prueba-tecnica-nodolab/

## 📁 Estructura del Proyecto
- `/index.html`: Estructura semántica HTML del componente (Versión A y B).
- `/scss/style.scss`: Archivo fuente de estilos estructurado con metodología BEM y variables CSS.
- `/css/style.css`: Archivo CSS final compilado y minificado.
- `/assets/img/`: Recursos gráficos y vectoriales extraídos de Figma.
- `/IA.md`: Bitácora de prompts e interacciones con la Inteligencia Artificial.

## 🚀 Instrucciones para ejecutar o visualizar el proyecto localmente
Para revisar el proyecto de forma local en tu computadora, sigue estos sencillos pasos:

1. **Descargar el proyecto:** Clona este repositorio usando Git (`git clone`) o descarga directamente el archivo ZIP desde el botón verde de GitHub y descomprímelo.
2. **Abrir el proyecto:** Navega hasta la carpeta raíz del proyecto (`prueba-tecnica-nodolab`).
3. **Visualizar en el navegador:** Haz doble clic sobre el archivo `index.html` para abrirlo directamente en cualquier navegador web moderno (Google Chrome, Mozilla Firefox, Safari, Edge, etc.). No requiere de ningún servidor local ni dependencias de Node.js para su visualización.
